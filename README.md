# Bathysphere — Reading Comprehension Question Sets

A browser-based tool that presents reading-comprehension questions for
well-known books across four levels of cognitive demand — **Literal,
Inferential, Analytical, and Evaluative** — each with an answer key,
follow-up questions, and an aligned CCSS ELA (Reading: Literature) standard.

This is a **static proof of concept**: it runs entirely in the browser with
no login and no API key. Pick a book and a grade band, and it displays a
ready-made question set for that text.

## Features
- A built-in selection of classic books, grouped by grade band
- Questions organized into four cognitive levels
- An answer key and two follow-up questions for every item
- A CCSS ELA Reading: Literature standard tag on each question
- Choose grade level and number of questions
- No account, no API key — open the page and use it

## Roadmap
An AI-powered version (built separately) connects live to the ~70,000 free
books in Project Gutenberg and generates fresh questions from any book — or
any pasted passage — using the Claude API and a small backend.
